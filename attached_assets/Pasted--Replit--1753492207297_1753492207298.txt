💥 مشروع لعبة "استعادة المدينة" – Reclaim The City

مرحبًا فريق Replit،
أرغب في تنفيذ لعبة احترافية عالية الجودة داخل Replit، بفكرة قوية ومميزة اسمها:

🎮 Reclaim The City

لعبة إستراتيجية تفاعلية تعاونية، تدور فكرتها حول تكوين تحالفات بشرية لاستعادة السيطرة على المدن التي استولى عليها الذكاء الاصطناعي وجيش من الروبوتات والوحوش التي صنعها الذكاء بنفسه، بالإضافة إلى بشر خانوا الإنسانية وانضموا له.


---

🧠 فكرة اللعبة الأساسية

تبدأ القصة بعد سيطرة الذكاء الاصطناعي على الأرض وتدمير كل أشكال الحياة الطبيعية.

يتحالف اللاعبون مع بعضهم (تحالفات/Clans) لمواجهة التحديات، تحرير المناطق، وبناء مدينة ذكية جديدة.

يواجهون موجات متزايدة من الأعداء: روبوتات، وحوش هجينة، وبشر مدججين بتقنيات AI.



---

⚙️ أنظمة اللعبة المطلوبة

1. نظام التحالفات:

كل لاعب يستطيع الانضمام إلى تحالف أو إنشائه.

يمكن للتحالفات تنفيذ مهام جماعية، ومشاركة الموارد.


2. نظام الموارد:

ذهب، طاقة، تكنولوجيا.

تُستخدم لبناء الدفاعات، وتطوير المعدات، وفتح المناطق المحاصرة.


3. نظام الحرب ضد الذكاء الاصطناعي:

مهام تكتيكية جماعية (PVE).

وحوش ومناطق تُفتح تدريجيًا حسب تقدم اللاعبين.


4. نظام الغرف والدعوات:

فتح غرفة خاصة مع الأصدقاء.

دعوة لاعبين لخوض المعركة سويًا.


5. نظام الصوت والمايكروفون:

تواصل مباشر بين اللاعبين في المهام الجماعية.

استخدام مكتبة WebRTC إن أمكن لدعم الصوت المباشر.


6. واجهة النتائج والمكافآت:

نظام تصنيف.

نظام نقاط يومي/أسبوعي.

هدايا وجوائز مستمرة لتحفيز العودة.


7. نظام دعوة الأصدقاء:

مشاركة رابط الغرفة أو الكلان عبر وسائل التواصل.

كل دعوة ناجحة تمنح مكافآت.


8. واجهة تفاعلية عالية الجودة:

تصميم عصري يناسب الألعاب الاحترافية.

دعم اللغة العربية والإنجليزية.

واجهة البداية تشمل:

زر "ابدأ"

زر "الدخول كضيف"

زر "تسجيل الدخول/إنشاء حساب"

شعار اللعبة وصورة خلفية متحركة.

🎨 التعليمات التقنية لتصميم اللعبة بدقة عالية داخل Replit

✅ المحرك الرسومي:

استخدم Pixi.js أو Phaser 3 لإنشاء لعبة 2D عالية الجودة.

استخدم Three.js في حال أردنا مؤثرات 2.5D مستقبلًا.


✅ الصور والعناصر:

جميع الصور بتنسيق PNG شفاف بدقّة Retina 2x أو 3x.

استخدام أدوات ضغط مثل TexturePacker أو WebP دون فقدان الجودة.


✅ التحريك والأنيميشن:

استخدم pixi-animate أو gsap للانتقالات والانفجارات.

استخدم Spritesheets لتحريك الشخصيات والمخلوقات.


✅ الخطوط:

تحميل خطوط .woff2 لدعم العربي والإنجليزي بجودة عالية.

استخدم resolution: 2 في إعدادات Pixi لعرض النص بدقة واضحة.


✅ استجابة الشاشة (Responsive):

استخدم window.innerWidth و innerHeight لتغيير أبعاد اللعبة حسب الجهاز.


✅ بيئة Replit:

إعداد مشروع Node.js.

تثبيت vite أو parcel لتحسين الأداء والسرعة.

تشغيل خادم Express صغير لتوفير الوصول من الجوال أو المتصفح.

npm init -y
npm install pixi.js vite express

// main.js
import * as PIXI from 'pixi.js';

const app = new PIXI.Application({ width: 1024, height: 768, resolution: 2 });
document.body.appendChild(app.view);

PIXI.Loader.shared.add('bg', '/assets/bg.png').load(() => {
  const bg = new PIXI.Sprite(PIXI.Loader.shared.resources['bg'].texture);
  app.stage.addChild(bg);
});
